# Install on Windows

The recommended path for most users is the portable Windows build from GitHub Releases.

## Steps

1. Download `pdf-toolkit-windows-x64.zip` from the latest release.
2. Extract the ZIP to a normal folder, not inside `Program Files`.
3. Open the extracted folder.
4. Run `pdf-toolkit-gui.exe`.

## What works immediately

- Merge, split, rotate, crop, preview, render, extract text, metadata, forms, attachments, tables, and automation workflows.

## Optional OCR

OCR is not bundled in the default public release yet. See [enabling-ocr.md](./enabling-ocr.md) if you want to add OCR support later.
