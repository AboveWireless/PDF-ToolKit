# Security Policy

Please do not report security issues in public GitHub issues.

## How to report

- Use GitHub private vulnerability reporting for this repository if it is enabled.
- If private reporting is not enabled yet, contact the maintainer privately through the repository owner's GitHub profile before sharing details.

## What to include

- A short description of the issue
- Impact and affected workflows
- Steps to reproduce
- Any proof-of-concept files or screenshots that help confirm the problem

We will acknowledge valid reports as quickly as practical and coordinate a fix before public disclosure.
